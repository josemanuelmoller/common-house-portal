Layout

Sidebar fijo de 228px. Content area con padding simétrico de 52px. Grids con gap de 16–24px.

/* Estructura base */ --sidebar-w: 228px; .main { margin-left: var(--sidebar-w); } .page-header { padding: 40px 52px 44px; background: var(--black); } .content { padding: 36px 52px 60px; } /* Grids de contenido */ .two-col { grid-template-columns: 1fr 1fr; gap: 20px; } .two-col-32 { grid-template-columns: 2fr 1fr; gap: 20px; } .three-col { grid-template-columns: repeat(3, 1fr); gap: 16px; } .stats-row { grid-template-columns: repeat(4, 1fr); gap: 12px; }

Reglas — No hacer

Restricciones del sistema. Violarlas rompe la coherencia visual.

✕
Usar colores fuera de la paleta de variables CSS
✕
Usar font-weight < 300 o > 900
✕
Border-radius > 14px en cards (excepto pills)
✕
Box-shadow — el sistema usa bordes, no sombras
✕
Fondo de color en cards — siempre --white o --bg
✕
Texto con opacity < 0.18 — ilegible
✕
Nuevas variantes de componente sin documentarlas aquí
✕
Padding de contenido > 52px horizontal
✕
--lime decorativo — solo para acciones / CTAs positivos
✕
Emojis como iconos en contextos formales o institucionales

Checklist — nueva vista

Verificar antes de finalizar cualquier vista nueva.

Header oscuro con background: var(--black)
Eyebrow en mayúsculas con la ruta (ej. Portfolio · Startups)
Título con patrón texto-light + <em> bold italic lime
Content area con padding: 36px 52px
Cards con border: 1.5px solid var(--border); border-radius: 14px
Labels ALL-CAPS en 8–9px / 700 / letter-spacing ≥ 1.5px
CTA principal en btn-lime
P1 banner al tope si hay señales críticas
Nav active state con border-left: 2px solid var(--lime)
Transiciones en todos los elementos interactivos (0.15s–0.2s)
Fondo de página es var(--bg) — no blanco
